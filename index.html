Moment 5 – Automatisering och publicering

I den här uppgiften har jag skapat en demowebbplats med HTML, CSS, JavaScript och bilder samt byggt en automatiserad arbetsprocess med Node.js-baserade verktyg. Syftet är att förenkla utvecklingen genom att automatisera uppgifter som bygg (build), minifiering och bildoptimering, och att publicera webbplatsen automatiskt via GitHub.

Det här innehåller projektet

  • Minst tre undersidor: startsida, bildsida och en process-sida.
  • Responsiv layout (fungerar på mobil, surfplatta och dator).
  • Automatiserad build med Vite (minifiering av CSS och JavaScript).
  • Automatiserad bildoptimering till moderna format (WebP/AVIF).
  • Automatisk publicering via GitHub Actions till GitHub Pages.

Så testar man

Under utveckling körs projektet lokalt med npm run dev. För att skapa en publiceringsversion används npm run build. Webbplatsen publiceras automatiskt när uppdateringar pushas till main-branchen.
